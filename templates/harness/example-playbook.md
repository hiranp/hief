## Agents Involved

| Agent-A (Planner) | Writes the feature spec using `hief docs generate spec` | CRDT fork from `main` |

| Human | Reviews merged intent graph and approves intents | N/A |

## Timeline

```

T=3:  Agent-A — commits spec to docs/specs/spec-search.md
T=4:  Agent-B — reads spec, runs `hief graph create --kind feature --title "Implement search"`
T=5:  Agent-B — updates intent status to `in_progress`
T=6:  CRDT merge: Agent-A ← Agent-B (intent graph merged)
T=7:  Verify: merged graph contains both agents' intents with distinct actor IDs
T=8:  Human — approves Agent-B's intent, sets status to `approved`
T=9:  Agent-B — begins implementation, files a golden eval case
```

- [ ] Both agents' intents exist in the merged document

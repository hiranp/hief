## Core Entities

### Intent

```rust
// From src/graph/intent.rs
pub struct Intent {
    pub id: String,           // hief-{8 hex chars}
    pub kind: String,         // feature | bug | refactor | spike | test | chore
    pub title: String,        // max 120 chars
    pub description: Option<String>,
    pub status: String,       // See status machine below
    pub priority: String,     // critical | high | medium | low
    pub assigned_to: Option<String>,
    pub created_at: DateTime<Utc>,
    pub updated_at: DateTime<Utc>,
}
```

**Invariants:**
- `id` must be globally unique — generated from UUIDv4, prefixed with `hief-`
- `status` must follow the defined state machine — invalid transitions are rejected
- `title` must never be empty

**Status Machine:**
```
draft → approved → in_progress → in_review → done
                                           ↘ rejected → revised → approved
```

### IndexChunk

```rust
// From src/index/chunker.rs
pub struct IndexChunk {
    pub id: String,       // blake3 hash of content
    pub file_path: String,
    pub symbol_name: Option<String>,
    pub symbol_kind: Option<String>, // function | struct | class | method
    pub language: String,
    pub content: String,
    pub line_start: u32,
    pub line_end: u32,
    pub indexed_at: DateTime<Utc>,
}
```

**Invariants:**
- `id` (content hash) must be recomputed on every update; stale chunks must be evicted
- `file_path` is always relative to `project_root`

## API Contracts (MCP Tools)

### Tool: `search_code`
- **Input:** `{ query: string, top_k?: number, language?: string, symbol_kind?: string }`
- **Output:** JSON string of `Vec<SearchResult>` — each has `file_path`, `symbol_name`, `snippet`, `score`
- **Errors:** Returns MCP `ErrorData` with kind `internal_error` on DB failure

### Tool: `create_intent`
- **Input:** `{ kind: string, title: string, description?: string, priority?: string, depends_on?: string }`
- **Output:** JSON string of the created `Intent`
- **Errors:** `invalid_params` if `kind` or `title` is invalid

### Tool: `get_project_context`
- **Input:** *(none)*
- **Output:** `{ index: IndexStats, active_intents: Intent[], ready_intents: Intent[] }`
- **Errors:** `internal_error` on DB failure

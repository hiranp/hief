# HIEF

Hybrid Intent‑Evaluation Framework (HIEF) is a lightweight Rust binary that provides
local indexing, agent intent tracking, and evaluation tooling for AI‑assisted
software engineering. This repository contains the HIEF implementation and its
supporting configuration.

## Getting Started

1. Install Rust (stable channel) and ensure `cargo` is on your PATH.
2. Run `cargo build` to compile the binary.
3. Start the MCP server with `hief serve` and use the provided CLI tools.

See `AGENTS.md` for workflow and agent instructions.

## Threat Model

Trust boundaries:

- query text to groundedness logic: Adversarial text can skew lexical overlap behavior.
- groundedness output to policy consumers: Miscalibrated score can cause bad gating decisions.

STRIDE register:

- T-03-05, Tampering, groundedness score input, mitigate: Normalize text and clamp score range to deterministic bounds.
- T-03-06, Repudiation, trajectory lineage, mitigate: Persist session_id, strategy, and timestamp for each event.
- T-03-SC, Tampering, cargo dependency changes, accept: No package-manager installs in this plan.

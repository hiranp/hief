## Threat Model

Trust boundaries:

- CLI args to installer command: Untrusted platform input may trigger unsafe file mutation intent.
- operation metadata to lane router: Misclassification can route sensitive operations to weak lane.

STRIDE register:

- T-02-03, Elevation of privilege, install command, mitigate: Validate platform enum strictly and default to dry-run for unknown states.
- T-02-04, Repudiation, route decisions, mitigate: Emit deterministic reason string for every selection.
- T-02-SC, Tampering, cargo dependency changes, accept: No package-manager installs in this plan.

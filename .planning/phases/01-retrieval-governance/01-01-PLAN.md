Introduce deterministic retrieval governance so HIEF search tools remain robust under context pressure.

## Tasks

### Task 1

Type: auto

TDD: true

Files: src/router/mod.rs, src/index/mod.rs, tests/test_search_routing.rs

Behavior:

- Symbol-style query (contains :: or short exact token) routes to deterministic strategy.
- Conceptual natural-language query routes to semantic strategy.
- Mixed query routes to hybrid strategy.

Action:

Create a pure routing API in src/router/mod.rs that classifies search queries into deterministic, hybrid, or semantic strategies per RET-01. Export router types from src/index/mod.rs or src/router/mod.rs where needed for tool-layer use. Add unit and integration tests in tests/test_search_routing.rs for classification and default fallback behavior. Do not couple this logic to MCP types; keep it reusable for CLI and MCP lanes.

Verify:

- automated: cargo test test_search_routing -- --nocapture

Done:

- Routing API is deterministic, covered by tests, and callable from tool handlers without MCP-specific coupling.

### Task 2

Type: auto

TDD: true

Files: src/mcp/tools.rs, src/index/search.rs, tests/test_search_routing.rs

Behavior:

- Response estimated over budget is truncated to configured boundary.
- Truncation preserves file path, symbol metadata, and snippet.

Action:

Add response-budget enforcement and compression for search-related MCP handlers in src/mcp/tools.rs per RET-02 and RET-03. Use a deterministic token estimator and preserve critical fields during compression. Add helper logic in src/index/search.rs only if necessary for snippet-safe truncation; avoid changing rank semantics. Add regression tests validating budget behavior and schema stability.

Verify:

- automated: cargo test test_search_routing -- --nocapture

Done:

- Search handlers return schema-stable, budget-constrained responses with no panics and tested truncation behavior.

## Threat Model

Trust boundaries:

- agent-client to MCP tools: Untrusted queries can trigger excessive compute and payload growth.
- MCP tools to index layer: Internal APIs may expose large raw content.

STRIDE register:

- T-01-01, Denial of service, search_code tool path, mitigate: Enforce max top_k, token budget, and early truncation in handler.
- T-01-02, Information disclosure, compressed payload logic, mitigate: Preserve only needed snippet context and avoid full-content overflow.
- T-01-SC, Tampering, cargo dependency changes, accept: No new package-manager install in this plan.
